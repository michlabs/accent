# accent
Package and CLI for removing Vietnamese accents from text

## accent CLI
Install and use:
```go
$ export GOPATH=$(pwd)
$ export PATH=$PATH:$GOPATH/bin
$ go get github.com/michlabs/accent/cmd/accent
$ accent help
$ accent remove -i input.txt -o output.txt
```
